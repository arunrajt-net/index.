How it works

1

Wipe & Log

Run secure wipe using NIST/DoD methods. Capture device metadata and a cryptographic hash of the wiped media.

2

Sign Certificate

Issuer signs the canonical certificate (JSON/PDF) using the private key (ECDSA P-256). Signature is saved as sidecar and embedded metadata.

3

Verify Publicly

A QR on the PDF links to a public verifier endpoint that checks signature, timestamp, and revocation list — instantly proving authenticity.
